## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## 🚀 Feature

- Login
- Register
- Direct messaging to other people
- Message in group
- Create group
- Ban User as admin group
- Send message with text, emoji, image and video
- Light/Dark mode
- Video call
- Chatting with AI
